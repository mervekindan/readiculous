# 📚 Book Tracker & Reading Progress App

A responsive and user-friendly React web application designed to guilt-trip you into finishing your reading list, cataloging books, visualizing stats, and earning badges. 

## 🌳 Live Demo
* **Vercel:** [[Click here](https://readiculous-app.vercel.app/)]

## 🌿 Features
* **Profile Gamification:** Profile customization modal allowing users to pick from a preset of cute animal avatars and set favourite genres.
* **Protected Application Routes:** Unauthenticated guests are met with a friendly access restriction message, keeping user profiles and reading progress completely private until authorization.
* **Book Cataloging:** Search and discover actual books dynamically via the OpenLibrary API.
* **Reading Progress:** Track books through different phases - "In Progress" and "Finished".
* **Streak:** Daily reading streak with built-in timer.
* **Challenges:** Challenges page to see your trophy cabinet. Turn those locked, gray achievements into shiny, colorized badges!
* **Fully Responsive UI:** Fully adaptive layout that shifts into an app-like experience on smartphones.

## 🧚 Tech Stack

* **Frontend:** React (JavaScript, React Router, Context API).
* **Styling:** Pure CSS (zero heavy UI frameworks because we love performance).
* **Mock Backend:** A RESTful JSON-Server acting as our database, surviving `PATCH`, `GET`, and `POST` requests.
* **Data Provider:** OpenLibrary Search API.

## 🚨 Mock Backend

* **Hosted on:** Render.com (Web Service)
> ⏳ **Note on Loading Time:** The backend is hosted on Render's free tier. If the website hasn't been visited recently, the server goes into "sleep mode." Please allow **30–40 seconds** for the initial request (like logging in or signing up) to wake up the server. After that, everything will work instantly!
